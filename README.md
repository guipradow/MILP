# MILP

The default pattern of a mixed integer linear programming is:  
  
$\min f(x, y)$  
subject to:$~~~~h(x, y) = 0~~~~m$ equations    
$~~~~~~~~~~~~~~~~~~~g(x, y)\leq 0~~~~r$ inequalities  
$~~~~~~~~~~~~~~~~~~~x \in \mathbf{R}^n~~~~~~~~~~n$ real variables  
$~~~~~~~~~~~~~~~~~~~y \in \left\{0, 1\right\}^d~~~d$ binary variables
